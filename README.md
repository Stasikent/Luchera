# Лучера

> Сопровождение специалиста лучевой диагностики на каждом этапе профессионального пути.

---

## Что такое Лучера?

Лучера — цифровая платформа для обучения, профессионального развития и ежедневной работы специалистов лучевой диагностики.

Платформа объединяет:

- обучение студентов;
- подготовку ординаторов;
- поддержку практикующих врачей;
- инструменты для рентгенолаборантов;
- возможности для преподавателей;
- современные технологии искусственного интеллекта.

---

## Почему появилась Лучера?

Современный специалист использует десятки разрозненных ресурсов:

- учебники;
- клинические рекомендации;
- законодательство;
- тесты;
- образовательные платформы;
- рабочие документы;
- научные статьи.

Лучера объединяет всё это в единую экосистему.

---

## Наша философия

Искусственный интеллект не заменяет специалиста.

Он помогает ему становиться сильнее.

---

## Статус проекта

Проект находится в активной разработке.

Первая цель — запуск MVP для кафедры лучевой диагностики.

---

## Основные направления

- Обучение
- Практика
- Профессиональное развитие
- Правовая поддержка
- AI-наставник
