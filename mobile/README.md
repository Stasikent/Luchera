# Mobile

В данной директории будет располагаться мобильное приложение платформы «Лучера».

## Назначение

Мобильное приложение позволит специалистам использовать основные возможности платформы в любое время и в любом месте.

## Целевая аудитория

- студенты;
- ординаторы;
- врачи-рентгенологи;
- рентгенолаборанты;
- преподаватели.

## Планируемые возможности

### Обучение

- просмотр учебных материалов;
- прохождение тестов;
- решение клинических случаев;
- просмотр прогресса обучения.

### Практика

- быстрый поиск клинических рекомендаций;
- просмотр нормативных документов;
- доступ к избранным случаям;
- поиск по базе знаний.

### Профессиональное развитие

- учет НМО;
- отслеживание сроков аккредитации;
- подбор курсов повышения квалификации;
- уведомления о новых образовательных программах.

### Уведомления

- новые задания;
- сообщения преподавателя;
- изменения законодательства;
- новые клинические рекомендации;
- напоминания о прохождении курсов и аккредитации.

- iOS
