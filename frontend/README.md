# Frontend

В этой директории будет располагаться клиентская часть платформы «Лучера».

## Назначение

Frontend отвечает за пользовательский интерфейс для:

- студентов;
- ординаторов;
- врачей-рентгенологов;
- рентгенолаборантов;
- преподавателей;
- администраторов.

## Планируемые возможности

- регистрация и авторизация;
- личный кабинет пользователя;
- просмотр учебных материалов;
- работа с клиническими случаями;
- прохождение тестов;
- взаимодействие с цифровым наставником;
- создание учебных групп и курсов;
- просмотр аналитики и результатов;
- доступ к профессиональным модулям платформы.

## Планируемый стек

- Next.js
- React
- TypeScript
- Tailwind CSS

Окончательный выбор UI-библиотеки и подхода к состоянию приложения будет зафиксирован отдельным архитектурным решением.

## Статус

Проектирование.
