# Книга проекта «Лучера»

> Сопровождение специалиста лучевой диагностики на каждом этапе профессионального пути.

Книга проекта является основным документом, описывающим концепцию, структуру, принципы и развитие экосистемы «Лучера».

Она предназначена для:

- участников команды;
- медицинских экспертов;
- преподавателей;
- представителей кафедр и образовательных организаций;
- потенциальных партнёров;
- разработчиков, присоединяющихся к проекту.

---

## Статус проекта

Проект находится на этапе проектирования и подготовки MVP.

Первая цель — запуск пилотной версии для кафедры лучевой диагностики с участием студентов и преподавателей.

# Основная идея

Лучера создаётся как единая цифровая экосистема для обучения, профессионального развития и ежедневной работы специалистов лучевой диагностики.

Экосистема должна сопровождать специалиста:

- во время обучения в университете;
- в ординатуре;
- при подготовке к аккредитации;
- в повседневной профессиональной работе;
- при повышении квалификации и переподготовке;
- в преподавательской и управленческой деятельности.

---

# Основной принцип

> **Лучера создаётся не ради технологий.**

> **Лучера создаётся ради специалистов, которые будут пользоваться ей каждый день.**

Каждое продуктовое решение должно отвечать на вопрос:

> Помогает ли оно специалисту учиться, работать или профессионально развиваться?

Если ответ отрицательный, решение не должно становиться частью текущего продукта.

---

# Принципы ведения книги

Книга проекта является живым документом.

Она обновляется по мере развития концепции, появления новых исследований и принятия продуктовых решений.

При этом:

- утверждённые главы не переименовываются без необходимости;
- новые решения фиксируются в `DECISION_LOG.md`;
- технические решения оформляются в ADR;
- идеи, не входящие в текущий релиз, переносятся в Backlog;
- история изменений сохраняется средствами Git.
