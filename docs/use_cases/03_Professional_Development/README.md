# Professional Development

Раздел описывает бизнес-процессы, связанные с непрерывным профессиональным развитием специалистов лучевой диагностики.

Профессиональное развитие сопровождает специалиста на протяжении всей карьеры: от окончания университета до работы в качестве опытного врача, преподавателя или руководителя подразделения.

Лучера помогает ориентироваться в возможностях дополнительного образования, отслеживать профессиональный прогресс, готовиться к аккредитации и формировать индивидуальную образовательную траекторию.

---

# Назначение

Раздел описывает процессы:

- поиска программ повышения квалификации;
- поиска программ профессиональной переподготовки;
- поиска образовательных мероприятий;
- учета образовательной активности;
- сопровождения НМО;
- подготовки к периодической аккредитации;
- хранения документов об образовании;
- формирования индивидуальной траектории развития.

# Границы раздела

В раздел **Professional Development** входят:

- дополнительное профессиональное образование;
- повышение квалификации;
- профессиональная переподготовка;
- сопровождение НМО;
- подготовка к аккредитации;
- хранение документов;
- рекомендации по развитию компетенций.

В раздел не входят:

- обучение студентов;
- управление учебными группами;
- кадровый учет;
- поиск работы;
- образовательные программы кафедры.

---

# Принципы

## Непрерывность

Профессиональное развитие рассматривается как непрерывный процесс на протяжении всей карьеры.

## Персонализация

Рекомендации формируются с учетом профессионального профиля пользователя.

## Актуальность

Информация о курсах, аккредитации и образовательных мероприятиях должна регулярно обновляться.

## Независимость

Лучера не является образовательной организацией, а предоставляет информацию и инструменты сопровождения.
