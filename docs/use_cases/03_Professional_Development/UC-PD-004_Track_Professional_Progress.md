# UC-PD-004

## Контроль профессионального прогресса

---

## Цель

Помочь специалисту отслеживать собственное профессиональное развитие, выполнение индивидуального плана, образовательную активность и достижение поставленных целей.

---

## Рабочая задача

Оценить, насколько успешно выполняется мой план профессионального развития.

---

## Основные участники

- врач-рентгенолог;
- рентгенолаборант;
- ординатор.

---

## Вторичные участники

- интеллектуальный помощник;
- модуль аналитики.

---

## Предусловия

- пользователь авторизован;
- существует индивидуальный план профессионального развития;
- в системе накоплены данные об образовательной активности пользователя.

---

## Триггер

Пользователь хочет оценить свой текущий прогресс профессионального развития.

---

## Основной сценарий

1. Пользователь открывает раздел профессионального развития.
2. Переходит к просмотру прогресса.
3. Система анализирует:
   - завершенные образовательные программы;
   - выполненные этапы плана;
   - накопленные образовательные активности;
   - приобретенные компетенции;
   - оставшиеся задачи.
4. Отображается текущий прогресс.
5. Пользователь изучает результаты.
6. При необходимости корректирует дальнейший план развития.

---

## Альтернативные сценарии

### Просмотр прогресса по отдельной цели

Пользователь анализирует прогресс только по выбранному направлению развития.

---

### Просмотр истории развития

Пользователь просматривает изменения профессионального профиля за выбранный период.

---

## Исключительные ситуации

### Недостаточно данных

Система сообщает, что информации недостаточно для формирования объективной оценки.

---

## Результат

Пользователь получает актуальную информацию о своем профессиональном прогрессе.

---

## Связанные сущности

- Пользователь;
- План профессионального развития;
- Компетенция;
- Образовательная программа;
- Профессиональная цель.

---

## Связанные модули

- Профессиональное развитие;
- Аналитика.

---

## Связанные Use Cases

- UC-PD-001 — Оценка профессиональных компетенций.
- UC-PD-002 — Формирование индивидуального плана профессионального развития.
- UC-PD-003 — Поиск программы профессионального развития.

---

## Ограничения

Прогресс рассчитывается только на основании доступных системе данных.

---

## Критерии успешности

- пользователь понимает текущий уровень достижения целей;
- отображается выполненная и оставшаяся часть плана;
- пользователь может скорректировать дальнейшее развитие.

---

## Возможности дальнейшего развития

- прогноз достижения целей;
- визуализация профессионального роста;
- сравнение нескольких планов развития;
- совместный контроль прогресса с наставником.
