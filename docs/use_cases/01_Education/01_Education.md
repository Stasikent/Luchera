# UC-EDU-001

## Создание учебного курса

### Цель

Создать новый курс для обучения студентов или ординаторов.

### Участники

- преподаватель;
- система.

### Предусловия

- преподаватель авторизован;
- обладает соответствующими правами.

### Основной сценарий

1. Создание курса.
2. Заполнение описания.
3. Добавление учебных тем.
4. Добавление материалов.
5. Добавление клинических случаев.
6. Добавление тестирования.
7. Публикация курса.

### Альтернативные сценарии

- сохранение черновика;
- копирование существующего курса.

### Исключительные ситуации

- отсутствуют права;
- курс с таким названием уже существует;
- превышен лимит вложений.

### Результат

Создан новый курс, готовый к назначению учебным группам.
