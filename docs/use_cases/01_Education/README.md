# Education

Раздел описывает образовательные процессы, которые дополняют существующую систему обучения учреждения.

Лучера не предназначена для обязательной замены университетской LMS, электронной ведомости или системы учета посещаемости.

Основная задача образовательного направления — предоставить специализированные инструменты для лучевой диагностики:

- работу с клиническими случаями;
- интерактивные задания;
- формирование описания и заключения;
- последовательные подсказки;
- разбор диагностических ошибок;
- самостоятельную подготовку;
- подготовку к экзаменам и аккредитации;
- дополнительную образовательную аналитику.

При необходимости результаты обучения могут экспортироваться или передаваться во внешнюю информационную систему учреждения.

---

# Назначение

Раздел описывает процессы:

- формирования образовательной программы;
- формирования учебного плана;
- создания учебных курсов;
- назначения курсов учебным группам;
- прохождения обучения;
- изучения отдельных тем;
- решения клинических случаев;
- прохождения тестирования;
- анализа результатов обучения;
- подготовки к экзаменам;
- подготовки к аккредитации.

# Границы раздела

В раздел **Education** входят:

- образовательные программы;
- учебные планы;
- учебные курсы;
- учебные материалы;
- задания;
- клинические случаи;
- тестирование;
- образовательная аналитика.

В раздел не входят:

- поиск вакансий;
- профессиональная деятельность врача;
- нормативно-правовая поддержка;
- административное управление учреждением;
- платформенные сервисы.

Эти процессы описываются в соответствующих разделах Use Cases.
