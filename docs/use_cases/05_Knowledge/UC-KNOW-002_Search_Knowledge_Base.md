# UC-KNOW-002

## Поиск материалов базы знаний

---

## Цель

Позволить пользователю быстро найти необходимую профессиональную информацию в базе знаний платформы.

---

## Рабочая задача

Найти материал, необходимый для решения профессиональной задачи.

---

## Основные участники

- врач;
- рентгенолаборант;
- ординатор;
- студент;
- преподаватель.

---

## Вторичные участники

- интеллектуальный помощник;
- поисковый сервис.

---

## Предусловия

- пользователь авторизован;
- база знаний содержит опубликованные материалы.

---

## Триггер

Пользователь хочет найти профессиональную информацию.

---

## Основной сценарий

1. Пользователь открывает раздел базы знаний.
2. Вводит поисковый запрос.
3. При необходимости применяет фильтры.
4. Система выполняет поиск.
5. Отображает список найденных материалов.
6. Пользователь открывает выбранный материал.
7. При необходимости переходит к связанным материалам.

---

## Альтернативные сценарии

### Поиск по категории

Пользователь ограничивает поиск выбранной категорией.

---

### Поиск по ключевым словам

Система учитывает ключевые слова и профессиональные термины.

---

### Интеллектуальный поиск

Система понимает смысл запроса и предлагает наиболее релевантные материалы.

---

## Исключительные ситуации

### Материалы не найдены

Система предлагает похожие материалы или рекомендует изменить поисковый запрос.

---

### Материал недоступен

Пользователь получает уведомление об отсутствии доступа.

---

## Результат

Пользователь получает доступ к материалам, соответствующим поисковому запросу.

---

## Связанные сущности

- Материал базы знаний;
- Категория;
- Поисковый запрос;
- Пользователь.

---

## Связанные модули

- База знаний;
- Интеллектуальная поддержка.

---

## Связанные Use Cases

- UC-KNOW-003 — Просмотр материалов по категориям.
- UC-KNOW-006 — Использование материалов базы знаний в других модулях.

---

## Ограничения

Поиск выполняется только по материалам, доступным пользователю.

---

## Критерии успешности

- материалы находятся быстро;
- результаты соответствуют запросу;
- пользователь может перейти к связанным материалам.

---

## Возможности дальнейшего развития

- поиск по медицинским изображениям;
- голосовой поиск;
- семантический поиск;
- поиск по содержимому документов;
- рекомендации похожих материалов.
