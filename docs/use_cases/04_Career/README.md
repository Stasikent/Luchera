# Career

Раздел описывает бизнес-процессы, связанные с профессиональной карьерой специалистов лучевой диагностики.

Карьерное сопровождение начинается с момента окончания обучения и продолжается на протяжении всей профессиональной деятельности специалиста.

Лучера помогает пользователю ориентироваться на рынке труда, анализировать собственные компетенции, находить подходящие вакансии, планировать профессиональное развитие и отслеживать карьерный рост.

---

# Назначение

Раздел описывает процессы:

- поиска вакансий;
- анализа рынка труда;
- формирования профессионального профиля;
- оценки соответствия вакансии;
- выявления недостающих компетенций;
- поиска возможностей профессионального роста;
- поиска международных возможностей трудоустройства.

# Границы раздела

В раздел **Career** входят:

- поиск вакансий;
- анализ требований работодателей;
- рекомендации по развитию компетенций;
- сопровождение профессионального роста.

В раздел не входят:

- кадровый учет;
- оформление трудовых отношений;
- расчет заработной платы;
- взаимодействие с государственными кадровыми системами.

---

# Принципы

## Независимость

Лучера не является кадровым агентством.

Платформа агрегирует информацию о вакансиях и помогает специалисту принимать решение.

## Актуальность

Информация о вакансиях должна регулярно обновляться.

## Персонализация

Рекомендации формируются с учетом профессионального профиля пользователя.

## Объективность

При анализе вакансий используются открытые требования работодателей без субъективной оценки.
