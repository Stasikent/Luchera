# UC-WORK-005

## Поиск рабочего шаблона

---

## Цель

Позволить специалисту быстро найти подходящий шаблон описания исследования или заключения для использования в ежедневной работе.

---

## Рабочая задача

Найти подходящий шаблон для выполнения исследования или оформления протокола.

---

## Основные участники

- врач-рентгенолог;
- рентгенолаборант.

---

## Вторичные участники

- поисковый сервис;
- база шаблонов.

---

## Предусловия

- пользователь авторизован;
- пользователю доступен модуль «Практика»;
- база шаблонов содержит опубликованные материалы.

---

## Триггер

Во время оформления исследования специалисту требуется готовый шаблон.

Например:

- описание исследования;
- заключение;
- структура протокола;
- стандартная формулировка;
- памятка по исследованию.

---

## Основной сценарий

1. Пользователь открывает каталог шаблонов.
2. Выполняет поиск по названию, области исследования или ключевым словам.
3. Система отображает список подходящих шаблонов.
4. Пользователь открывает выбранный шаблон.
5. Просматривает содержимое.
6. При необходимости копирует шаблон или открывает его для дальнейшего использования.

---

## Альтернативные сценарии

### Поиск по анатомической области

Пользователь выбирает область исследования.

Система отображает соответствующие шаблоны.

---

### Поиск по модальности

Пользователь выбирает:

- рентгенография;
- КТ;
- МРТ;
- маммография;
- флюорография;
- денситометрия;
- другое.

---

### Поиск по заболеванию

Пользователь ищет шаблоны, связанные с конкретной патологией.

---

## Исключительные ситуации

### Подходящий шаблон отсутствует

Система сообщает об отсутствии результатов и предлагает похожие шаблоны.

---

### Шаблон находится в архиве

Система предупреждает пользователя и предлагает актуальную версию.

---

## Результат

Пользователь получает подходящий рабочий шаблон и может использовать его при оформлении исследования.

---

## Ограничения

Система предоставляет шаблоны как основу для оформления исследования.

Окончательное описание и заключение формируются специалистом.

---

## Критерии успешности

- пользователь быстро находит необходимый шаблон;
- шаблон соответствует выбранной модальности;
- шаблон содержит актуальную структуру оформления;
- пользователь может сразу использовать шаблон в своей работе.

---

- автоматический подбор шаблона на основании типа исследования;
- совместное редактирование шаблонов.
