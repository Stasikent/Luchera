# UC-WORK-001

## Поиск профессионального материала

---

## Цель

Позволить специалисту быстро найти профессиональный материал, необходимый для выполнения текущей рабочей задачи.

---

## Основные участники

- врач-рентгенолог;
- рентгенолаборант;
- руководитель подразделения.

---

## Вторичные участники

- поисковый сервис;
- база знаний платформы.

---

## Предусловия

- пользователь авторизован;
- пользователь имеет доступ к модулю практики;
- в базе знаний присутствуют доступные профессиональные материалы.

---

## Триггер

Специалисту требуется уточнить информацию, связанную с профессиональной деятельностью.

Примеры:

- признаки заболевания;
- методика исследования;
- параметры выполнения исследования;
- алгоритм действий;
- практическая памятка;
- профессиональная статья;
- инструкция;
- справочная таблица.

---

## Основной сценарий

1. Пользователь открывает поиск.
2. Вводит поисковый запрос.
3. Система выполняет поиск по доступным профессиональным материалам.
4. Система отображает список результатов.
5. Пользователь применяет фильтры при необходимости.
6. Пользователь открывает выбранный материал.
7. Система отображает содержимое, источник и дату актуальности.
8. Пользователь при необходимости сохраняет материал в избранное.

---

## Альтернативные сценарии

### Поиск по тематической категории

1. Пользователь открывает каталог профессиональных материалов.
2. Выбирает раздел или тему.
3. Система отображает связанные материалы.
4. Пользователь открывает выбранный материал.

### Уточнение поискового запроса

1. Система не находит точного совпадения.
2. Система предлагает уточнить запрос или выбрать похожую тему.
3. Пользователь изменяет запрос.
4. Система повторяет поиск.

### Фильтрация результатов

Пользователь ограничивает результаты по:

- типу материала;
- тематике;
- методу исследования;
- дате публикации;
- дате актуальности;
- источнику.

---

## Исключительные ситуации

### Результаты отсутствуют

Система сообщает, что материалы по запросу не найдены, и предлагает:

- изменить формулировку запроса;
- убрать часть фильтров;
- перейти к связанным темам;
- отправить предложение о добавлении материала.

### Материал недоступен

Система сообщает об ограничении доступа.

### Материал устарел

Система отображает предупреждение о возможной неактуальности материала и указывает дату последней проверки.

---

## Результат

Пользователь получает доступ к профессиональному материалу, соответствующему его запросу.

При необходимости материал сохраняется в избранное или истории просмотра.

## Ограничения

- сценарий не включает формирование ответа интеллектуальным помощником;
- сценарий не включает поиск по медицинскому изображению;
- сценарий не включает поиск нормативных документов;
- сценарий не включает поиск похожего клинического случая;
- сценарий не включает автоматическую интерпретацию медицинских данных.

---

## Критерии успешности

- пользователь может сформулировать запрос без знания внутренней структуры базы;
- результаты соответствуют запросу;
- пользователь видит источник и дату актуальности материала;
- нужный материал доступен за минимальное количество действий;
- система корректно обрабатывает отсутствие результатов.

---

## Возможности дальнейшего развития

- персонализированное ранжирование;
- сохранённые поисковые запросы;
- рекомендации связанных материалов;
- история поиска;
- голосовой ввод запроса;
- поиск по синонимам и профессиональной терминологии.
