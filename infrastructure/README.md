# Infrastructure

В данной директории располагается инфраструктура платформы «Лучера».

## Назначение

Infrastructure отвечает за развертывание, сопровождение, безопасность и масштабирование платформы.

Главная цель — обеспечить надежную работу всех компонентов системы независимо от среды выполнения.

---

## Планируемые компоненты

### Контейнеризация

- Docker
- Docker Compose

---

### Базы данных

- PostgreSQL
- Redis

---

### Хранение файлов

Рассматриваемые варианты:

- S3
- MinIO
- Local Storage (для разработки)

---

### CI/CD

Планируется использование:

- GitHub Actions

В дальнейшем возможно:

- GitLab CI
- Azure DevOps

---

### Мониторинг

Планируется:

- Prometheus
- Grafana

---

### Логирование

Планируется:

- Loki
- Grafana
- Structured Logging

---

### Резервное копирование

Будут реализованы:

- автоматическое резервное копирование базы данных;
- резервное копирование пользовательских файлов;
- контроль восстановления данных.

---

### Безопасность

Планируется:

- HTTPS
- JWT
- RBAC (Role-Based Access Control)
- аудит действий пользователей
- журнал безопасности

---

## Среды разработки

Планируется поддержка:

- Local Development
- Testing
- Staging
- Production
